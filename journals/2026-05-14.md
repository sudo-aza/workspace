### Technical details
- Script reads git `--numstat` for per-commit line change data
- Parses journal markdown files to extract task completion counts per run
- Parses todo.md for pending/done status
- Uses matplotlib with a custom color palette matching the sudodoc theme
- All plots saved as PNG at 150 DPI
- Also generates a `summary.txt` with key metrics
