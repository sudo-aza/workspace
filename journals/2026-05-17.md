### Key lesson from OmniVoice install

PyTorch default install pulls CUDA deps (~900 MB including nvidia-cudnn 366 MB). Use `--index-url https://download.pytorch.org/whl/cpu` for CPU-only (~200 MB). Saves 700 MB on a 10 GB disk.
